## 1. 核心问题

Phase 1-3 已将 AI-era 短时链条闭合，但缺历史零点：

> AI-era 位移是几百年围棋 Codex 漂移中的正常波动，还是超出历史 envelope 的新 regime？

Phase 0.5 回答前半句。Phase 4 进一步追问：CWI 数据自身的构成变化是否解释了观测到的信号？以及 Codex 的流通速度是否在 AI 时代发生了变化？

## 2. Phase 0.5 — 长历史漂移封套

### 方法

- 按 decade 切分 1600-2026 全部年份数据
- 每十年计算 SHP rhythm centroid (5 维)
- 十年间 drift 向量
- Pre-2016 95% 漂移封套
- AI-era (2010s→2020s) 位移 vs 封套

### 结果

```
Pre-2016 decade drift:  mean |drift| = 0.0146 ± 0.0098
AI-era (2010s→2020s):   |drift| = 0.0130
Z-score: -0.16
→ AI shift WITHIN historical envelope
```

历史上最大的漂移是 1910s→1920s (|drift|=0.034)，是 AI 时代漂移的 2.6 倍。AI 不是围棋史上最剧烈的结构变迁。

### 五年特征位移

| Feature | 2000s | 2020s | Δ | 超出封套? |
|---------|-------|-------|-----|----------|
| adj_opp | 0.1432 | 0.1507 | +0.0075 | no |
| adj_own | 0.1207 | 0.1211 | +0.0005 | no |
| dens_delta | -0.0291 | -0.0334 | -0.0043 | no |
| dist_last | 0.1804 | 0.1794 | -0.0010 | no |
| is_corner_open | 0.1371 | 0.1506 | +0.0135 | marginal |

### Codex Δ 跨十年

Human Codex 在所有可测的十年间始终超过 Frequency baseline，均值 Δ = +0.027 bits/move。Codex operation 是跨时代的普遍属性。

## 4. Phase 4A — 档案构成矩阵

### 方法

建立 `decade × region × player × event × source` 的档案构成矩阵。

### 关键发现

1. **地域熵在 2010s 达峰 (1.086)**，2020s 略降至 0.830
2. **棋手多样性在 1960s 爆炸**：149→388 人，Top5 集中度从 0.445→0.104
3. **赛事多样性同步爆炸**：1930s 103 赛事 → 2010s 841 赛事
4. **1910s 之前**：档案以"Castle Game"和"10-game match"为主——私人记录，非制度赛事
5. **收录断点**：1933→1934 (5.3×)、1959→1960 (3.8×)，对应 Oteai 制度化与现代赛事体系建立

### 解释

CWI 不是围棋历史的透明窗口。它是围棋 Codex 外部记忆网络的沉积物。档案构成的变化本身就是 Codex 演化的证据。

## 5. Phase 4B — 区域控制复验

### 核心问题

Phase 0.5 和 Phase 1-3 的信号是否由 CWI 的地域构成变化（中韩棋谱进入）驱动？

### 方法

三个档案视图下复验 Phase 0.5 的 decade drift：
- Full CWI（基线）
- Japan-only（最长连续档案，N=426K+）
- Region-balanced（每地区每十年等量下采样）

### 结果

```
2000s→2020s |drift|:
  Full CWI:          0.0160
  Japan-only:        0.0159  ← 与 Full 几乎相同
  Region-balanced:   0.0277  ← 更强, 非更弱
```

### 结论

信号在棋步里，不在档案构成里。Japan-only 完全保留了 Full 的信号。区域平衡放大而非消除信号。

## 6. Phase 4C — 1910s→1920s 拆解

### 核心问题

历史上最大的十年漂移 (|drift|=0.034，是 AI 时代的 2.6 倍) 是真实的棋步变化，还是档案构成变化？

### 方法

将 1910s (150 局) 和 1920s (412 局) 拆分为：
- 重叠棋手 (20 人同时在两十年有对局)
- 非重叠棋手 (仅在一个十年出现)

### 结果

```
|drift| Full:              0.0342
|drift| Overlap players:   0.0291  ← 85% 来自同一批棋手
|drift| Non-overlap:       0.0292
```

### 1910s vs 1920s 的档案构成

- 1910s：147/150 局事件字段为空（私人对局）
- 1920s：Oteai 制度建立（1928），职业排名赛事出现
- 棋手池从 43 人→110 人（2.6× 扩张）

### 结论

1910s→1920s 漂移以真实棋步变化为主（重叠棋手保留了 85%），伴随 Oteai 制度革命。这是**制度驱动的 Codex 重组**——与 AI 时代的 Oracle 驱动重组机制不同。

### 两场革命对比

| | 1910s→1920s 制度革命 | 2010s→2020s Oracle 革命 |
|---|---|---|
| 驱动因素 | Oteai 职业排名制度建立 | AI (KataGo) policy oracle |
| 棋手池变化 | 43→110 (2.6×) | 保持稳定 |
| 棋步漂移幅度 | 0.034 (历史最大) | 0.013 (历史均值水平) |
| 档案构成变化 | 私人记录→制度记录 | 日本为主→多区域 |
| Codex 机制 | 制度重组 | Oracle 对齐 |

## 7. Phase 4D — Pattern 传播与存活

### 核心问题

Codex 的流通速度——而非棋步结构——是否在 AI 时代发生了变化？

### 方法

- 定义 pattern: 前 15 手开局序列 (74,501 个不重复 pattern, 546 个满足频率门槛)
- 计算每个 pattern 的首次出现、采纳延迟、存活半衰期、跨区域延迟
- 按 Era 分层比较
- Held-out 检验: 历史频率是否预测来年采纳？

### 结果

```
                     pre-1980  1980-1999  2000-2015  2016+
Adoption lag (年):       6.3       5.7        2.6      1.5
Reuse rate:             0.35      0.37       0.67     0.78
Half-life (年):          1.1       1.1        1.0      1.1
Cross-region lag (年):   —         —          —        —
```

**Adoption lag 下降 4.2×**——新 pattern 被采纳的速度从 6.3 年缩短到 1.5 年。

**Reuse rate 上升 2.2×**——一旦采纳，pattern 被锁定的概率从 35% 升至 78%。

**Codex Δ = +0.081 (2× over random):** 历史 pattern 频率预测来年采纳的能力是随机基线的 2 倍。

### 最持久 pattern

跨越 47-57 年的 pattern 均为日本古典开局（星-小目组合），是围棋的"定式化石"。

### 结论

AI 时代改变了 Codex 的流通速度，而非棋步结构。Pattern 采纳加速和锁定率提高是流通结构变化的最干净信号。

## 8. Phase 4E — 区域 Rhythm 与 Oracle 收敛

### 区域 rhythm 签名

1990-2015 的区域 rhythm 签名显示三个清晰操作点：

| Region | N | adj_opp | is_corner_open | dist_last | 解释 |
|--------|------:|--------:|---------------:|----------:|------|
| JPN | 949K | 0.1421 | 0.1326 | 0.1794 | 厚实 / 地域型 |
| KOR | 95K | 0.1454 | 0.1422 | 0.1825 | 战斗 / 灵活型 |
| CHN | 18K | 0.1484 | 0.1464 | 0.1834 | 战斗 + 高机动型 |

日本最不贴对手、最少开新角、步伐最连贯；韩国更贴对手、更灵活；中国在韩国基础上更极端。

### AI 三阶段区域距离

```
1990s:   0.0186  最分散：三个独立传统
2016-17: 0.0070  最收敛：AlphaGo 冲击
2018-21: 0.0121  重新分化：各自吸收 AI
2022-25: 0.0065  再次收敛：Oracle Codex 稳定
```

AI 不是简单抹平区域风格，而是产生：

```text
shock convergence -> differentiated absorption -> oracle stabilization
```

### 韩国战斗指数

```
1990s:   +0.018  李昌镐时代，最“日本化”
2008-15: +0.027  李世石巅峰
2018-21: +0.038  AI 扩散期，战斗精神达到历史最高
2022-25: +0.032  Oracle 时代，开始回落
```

AI 最初放大了韩国的战斗风格，而不是抹平它；Oracle 时代才开始拉回。

### 与 Phase 3 的对应

区域三阶段独立复现了 Phase 3 的 `E1 peak -> E2 trough -> E3 recovery`：

| Phase 3 | 区域视角 | 解释 |
|---------|----------|------|
| E1 peak | 2016-17 区域距离最低 | AlphaGo shock 造成共同收敛 |
| E2 trough | 2018-21 区域重新分化 | 各区域以自身风格吸收 AI，历史 Codex 暂时退化 |
| E3 recovery | 2022-25 再次收敛 | Oracle Codex 稳定，新操作点重新沉积 |

## 9. 综合叙事

```
Phase 0.5:  AI-era rhythm shift 在世纪尺度历史封套内
            → AI 不是断裂，是延续

Phase 0.5+: 棋谱流通结构发生了革命
            → CWI 自身从中韩棋谱为零变为全球档案

Phase 4A:   CWI 是 Codex 沉积物，不是透明窗口
            → 档案构成变化本身就是 Codex 演化的证据

Phase 4B:   信号在棋步里，不在档案构成里
            → Japan-only 完全复现 Full CWI 结果

Phase 4C:   1910s→1920s 是制度革命 (Oteai)，2010s→2020s 是 Oracle 革命 (AI)
            → 不同驱动机制，不同可见层级；AI 不以最大棋步漂移表现

Phase 4D:   AI 时代改变了 Codex 流通速度
            → Pattern 采纳 4.2× 加速，锁定率 2.2× 提高

Phase 4E:   AI 没有简单抹平区域风格
            → shock 收敛、分化吸收、oracle 稳定
```

### 一句话

> AI 没有重写围棋的棋步结构——它重写了围棋 Codex 被发现、传播和锁定的速度；它也没有简单抹平区域风格，而是先制造共同震荡，再放大区域性吸收，最后让 Oracle Codex 稳定收敛。
